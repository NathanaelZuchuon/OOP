				BankAccountCode Overview

This code allows virtual money management.

The user can:

	1. Create his account.
	2. Be connected to his account.
	3. Make transfert.
	4. Delete his account.

Once connected to his account, he can make:

	1. Deposits.
	2. Withdrawals.
	3. Consultations.

When disconnected, the history of these transactions will be displayed.
